# VDMX Risk Intelligence - Backend MVP

Backend mínimo para soporte de pagos y formularios.

## Instalación
```bash
npm install
```

## Ejecución
```bash
npm start
```

## Endpoints de prueba

- `GET /health` - Estado general del servidor
- `GET /api/payments/ping` - Verificar endpoint de pagos
- `GET /api/payments/health` - Estado de tabla payments
- `GET /api/forms/ping` - Verificar endpoint de formularios
- `GET /api/forms/health` - Estado de tabla forms

## Stack

- Node.js
- Express
- SQLite3
- CORS
- Body-parser
- Dotenv
